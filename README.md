# titotuteAPP

## Registra tu rendimiento

Aplicación web para registrar y hacer seguimiento de tu rendimiento académico.

### Características

- **Registro de sesiones de estudio**: Guarda información detallada de cada sesión incluyendo asignatura, tema, duración, ejercicios completados y calificaciones.
- **Estadísticas en tiempo real**: Visualiza el número total de sesiones, tiempo total estudiado y promedio general de calificaciones.
- **Historial completo**: Revisa todas tus sesiones anteriores con detalles completos.
- **Almacenamiento local**: Todos los datos se guardan localmente en tu navegador.

### Cómo usar

1. Abre `index.html` en tu navegador web
2. Completa el formulario "Registrar Sesión de Estudio" con:
   - Asignatura
   - Tema estudiado
   - Duración en minutos
   - Número de ejercicios completados
   - Calificación (0-10)
   - Notas adicionales (opcional)
3. Haz clic en "Guardar Rendimiento"
4. Visualiza tu historial y estadísticas en el panel derecho

### Tecnologías

- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage API
